## ✨ Funktionen

- **✅ Inline-Injektion:** Fügt Kontrollkästchen neben Ihrem Chat-Verlauf direkt auf `gemini.google.com` ein.
- **💥 Massenlöschung:** Wählen Sie mehrere unerwünschte Chats aus, um sie alle auf einmal zu löschen.
- **🤖 Automatisiert:** Überspringt die manuellen Bestätigungs-Popups und erledigt die UI-Klicks für Sie.
- **🌍 Globale Unterstützung:** Nativ lokalisiert für Englisch, Spanisch, Französisch, Deutsch, Portugiesisch, Italienisch, Japanisch und Chinesisch.
- **🛡️ Kugelsicheres Polling:** Integrierter DOM-Poller, der die komplexe Architektur von Gemini problemlos überlebt.
- **⚡ Optimiert:** Wird mit perfekt skalierten nativen Symbolen für ein minimales Paketgewicht geliefert.

## 🚀 Installation (Entwicklung)

1. Laden Sie das neueste `.zip` von der **[GitHub Releases](https://github.com/JLDesignNetwork/gemini-chat-nuke-ff/releases)** Seite herunter.
2. Öffnen Sie Firefox und navigieren Sie zu `about:addons`.
3. Klicken Sie auf das ⚙️ **Zahnrad-Symbol** und wählen Sie **Add-on aus Datei installieren...**
4. Wählen Sie Ihr heruntergeladenes `.zip` Archiv aus. Fertig!

## 📦 Verpackung

```bash
./package.sh 1.2.0 beta
```
