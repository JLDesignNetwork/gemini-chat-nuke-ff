## ✨ Recursos

- **✅ Injeção Inline:** Insere caixas de seleção ao lado do seu histórico de chat diretamente em `gemini.google.com`.
- **💥 Exclusão em Massa:** Selecione vários chats indesejados para excluí-los de uma só vez.
- **🤖 Automatizado:** Ignora os pop-ups de confirmação manual e gerencia os cliques da interface para você.
- **🌍 Suporte Global:** Localizado nativamente para inglês, espanhol, francês, alemão, português, italiano, japonês e chinês.
- **🛡️ Sondagem à prova de falhas:** Construído com um verificador DOM resiliente que sobrevive à arquitetura SPA do Gemini.
- **⚡ Otimizado:** Vem com ícones nativos perfeitamente dimensionados para renderização nítida.

## 🚀 Instalação (Desenvolvimento)

1. Baixe o `.zip` mais recente na página **[Releases do GitHub](https://github.com/JLDesignNetwork/gemini-chat-nuke-ff/releases)**.
2. Abra o Firefox e acesse `about:addons`.
3. Clique no ⚙️ **ícone de engrenagem** e selecione **Instalar de um Arquivo...**
4. Selecione o arquivo `.zip` baixado. Pronto para excluir!
