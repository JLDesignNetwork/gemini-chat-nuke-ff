## ✨ Fonctionnalités

- **✅ Injection en ligne:** Injecte des cases à cocher à côté de votre historique de chat directement sur `gemini.google.com`.
- **💥 Suppression en masse:** Sélectionnez plusieurs discussions indésirables pour les supprimer toutes en même temps.
- **🤖 Automatisé:** Ignore les fenêtres de confirmation manuelles fastidieuses et gère les clics d'interface pour vous.
- **🌍 Support mondial:** Localisé nativement pour l'anglais, l'espagnol, le français, l'allemand, le portugais, l'italien, le japonais et le chinois.
- **🛡️ Sondage infaillible:** Construit avec un analyseur DOM résilient qui survit à l'architecture de la page de Gemini.
- **⚡ Optimisé:** Livré avec des icônes natives WebExtension parfaitement dimensionnées.

## 🚀 Installation (Développement)

1. Téléchargez la dernière version `.zip` depuis la page **[GitHub Releases](https://github.com/JLDesignNetwork/gemini-chat-nuke-ff/releases)**.
2. Ouvrez Firefox et accédez à `about:addons`.
3. Cliquez sur ⚙️ **l'icône d'engrenage** et sélectionnez **Installer un module depuis un fichier...**
4. Sélectionnez votre archive `.zip`. C'est prêt !
