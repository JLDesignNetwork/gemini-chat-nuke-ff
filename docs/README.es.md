## ✨ Características

- **✅ Inyección en línea:** Inyecta casillas de verificación junto a tu historial de chat directamente en `gemini.google.com`.
- **💥 Eliminación Masiva:** Selecciona múltiples chats no deseados para eliminarlos todos a la vez.
- **🤖 Automatizado:** Omite las tediosas confirmaciones manuales y maneja los clics de la interfaz por ti.
- **🌍 Soporte Global:** Localizado nativamente para inglés, español, francés, alemán, portugués, italiano, japonés y chino.
- **🛡️ Sondeo a prueba de balas:** Construido con un sondeador DOM resistente que sobrevive a la arquitectura de la página de Gemini.
- **⚡ Optimizado:** Se envía con iconos nativos perfectamente escalados para una representación nítida y un peso mínimo.

## 🚀 Instalación (Desarrollo)

1. Descarga la última versión `.zip` de la página de **[Lanzamientos de GitHub](https://github.com/JLDesignNetwork/gemini-chat-nuke-ff/releases)**.
2. Abre Firefox y navega a `about:addons`.
3. Haz clic en el ⚙️ **icono de engranaje** y selecciona **Instalar complemento desde un archivo...**
4. Selecciona tu archivo `.zip` descargado. ¡Listo para eliminar!

## 📦 Empaquetado
