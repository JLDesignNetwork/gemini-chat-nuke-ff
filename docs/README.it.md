## ✨ Caratteristiche

- **✅ Iniezione in linea:** Inietta caselle di controllo accanto alla cronologia delle chat direttamente su `gemini.google.com`.
- **💥 Eliminazione di massa:** Seleziona più chat indesiderate per eliminarle tutte in una volta.
- **🤖 Automatizzato:** Salta i noiosi popup di conferma manuale e gestisce i clic dell'interfaccia al posto tuo.
- **🌍 Supporto Globale:** Localizzato nativamente per inglese, spagnolo, francese, tedesco, portoghese, italiano, giapponese e cinese.
- **🛡️ Polling infallibile:** Costruito con un polling DOM resiliente che sopravvive all'architettura complessa di Gemini.
- **⚡ Ottimizzato:** Viene fornito con icone native WebExtension perfettamente dimensionate.

## 🚀 Installazione (Sviluppo)

1. Scarica l'ultima versione `.zip` dalla pagina **[Release di GitHub](https://github.com/JLDesignNetwork/gemini-chat-nuke-ff/releases)**.
2. Apri Firefox e vai su `about:addons`.
3. Fai clic sull'icona ⚙️ **ingranaggio** e seleziona **Installa componente aggiuntivo da file...**
4. Seleziona l'archivio `.zip` scaricato. Pronto!

## 📦 Confezionamento

./package.sh 1.2.1 beta
