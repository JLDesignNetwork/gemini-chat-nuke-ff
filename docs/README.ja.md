## ✨ 特徴

- **✅ インライン挿入:** `gemini.google.com` のチャット履歴の横にチェックボックスを直接挿入します。
- **💥 一括削除:** 不要なチャットを複数選択し、一度にすべて「削除」します。
- **🤖 自動化:** 面倒な手動確認ポップアップをスキップし、UIクリックを自動で処理します。
- **🌍 グローバルサポート:** 英語、スペイン語、フランス語、ドイツ語、ポルトガル語、イタリア語、日本語、中国語にネイティブ対応。
- **🛡️ 確実なポーリング:** Geminiの複雑なSPAルーティングにも耐える堅牢なDOMポーラーを搭載。
- **⚡ 最適化:** 完璧なサイズのネイティブアイコンにより、軽量化と鮮明な描画を実現。

## 🚀 インストール

1. **[GitHub Releases](https://github.com/JLDesignNetwork/gemini-chat-nuke-ff/releases)** ページから最新の `.zip` リリースをダウンロードします。
2. Firefoxを開き、URLバーに `about:addons` と入力します。
3. 右上の ⚙️ **歯車アイコン** をクリックし、**ファイルからアドオンをインストール...** を選択します。
4. ダウンロードした `.zip` ファイルを選択します。これで完了です！

## 📦 パッケージング

```bash
./package.sh 1.3.0 beta
```
